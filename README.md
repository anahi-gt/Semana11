Este proyecto consiste en un sistema avanzado de gestión de inventarios que desarrollé aplicando los principios de la Programación Orientada a Objetos (POO) en Python. El objetivo principal fue crear una herramienta capaz de administrar productos de forma eficiente, permitiendo realizar operaciones de registro, eliminación, actualización y búsqueda mediante una interfaz interactiva en la consola.
Para la gestión de los datos, decidí utilizar un diccionario como colección principal. Esta elección me permitió organizar los productos usando su ID único como clave, lo que garantiza que las búsquedas y modificaciones sean sumamente rápidas. Además, implementé un sistema de persistencia de datos mediante un archivo de texto llamado inventario.txt.
